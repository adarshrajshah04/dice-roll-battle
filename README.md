# 🎲 Dice Duel

A simple 2-player dice game built using HTML, CSS, and JavaScript.

## 🚀 Features

* 👥 Two-player game
* 🎲 Random dice roll system
* 🏆 Winner announcement after each round
* 📊 Live score tracking
* 📱 Responsive design (mobile + desktop)

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## 🎮 How to Play

1. Enter both player names
2. Click on **Login**
3. Press the **Roll** button
4. The player with the higher number wins the round
5. Scores update automatically
